VoxOver: AI Narration Studio - Setup Instructions

This guide will walk you through setting up and running the VoxOver AI Narration Studio locally on your computer.

Note: This guide assumes you already have Python and VS Code installed on your computer.

1. Install FFmpeg

FFmpeg is required for video and audio processing. Installation varies by operating system:

Windows

  1. Download FFmpeg from https://www.gyan.dev/ffmpeg/builds/ (the "release essentials" package)
  2. Extract the ZIP file to a location on your computer (e.g., C:\FFmpeg)
  3. Add FFmpeg to your system PATH:
    • Search for "Environment Variables" in the Start menu and select "Edit the system environment variables"
    • Click the "Environment Variables" button
    • Under "System Variables", find the "Path" variable, select it, and click "Edit"
    • Click "New" and add the path to the FFmpeg bin directory (e.g., C:\FFmpeg\bin)
    • Click "OK" on all dialogs to save your changes
  4. Verify the installation by opening a new Command Prompt and typing: 
    ffmpeg -version

macOS

Using Homebrew (recommended):

  1. If you don't have Homebrew installed, install it first: 
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Install FFmpeg: 
    brew install ffmpeg
  3. Verify the installation: 
    ffmpeg -version

Linux (Ubuntu/Debian)

  1. Update your package list: 
    sudo apt update
  2. Install FFmpeg: 
    sudo apt install ffmpeg
  3. Verify the installation: 
    ffmpeg -version

2. Create GitHub Repository (Optional)

  1. Go to GitHub and sign in to your account
  2. Click on the "+" icon in the top-right corner and select "New repository"
  3. Name your repository (e.g., "voxover-ai-narration-studio")
  4. Add an optional description
  5. Choose whether to make it public or private
  6. Click "Create repository"

3. Clone Repository in VS Code

  1. Open VS Code
  2. Press Ctrl+Shift+P (Windows/Linux) or Cmd+Shift+P (macOS) to open the command palette
  3. Type "Git: Clone" and select the command
  4. Paste your repository URL or select the repository from the list if you're signed in to GitHub in VS Code
  5. Select a folder on your local machine where you want to clone the repository
  6. Click "Open" when prompted to open the cloned repository

If you skipped creating a GitHub repository, simply create a new folder on your computer and open it in VS Code.

4. Add Project Files

  1. Download the following files from the class website:
    • utils.py
    • genai.py
    • requirements.txt
    • ai_prompt.txt
  2. Add these files to your project folder (the one you opened in VS Code)
  3. Create a new file named .env in your project folder
  4. Add your OpenAI API key to the .env file: 
    OPENAI_API_KEY=your_openai_api_key_here
  5. Install required packages using pip: 
    pip install -r requirements.txt
  6. Create a temp folder in your project directory: 
    mkdir temp

5. Create app.py Using AI

  1. Open the ai_prompt.txt file to view the prompt
  2. Copy the content of this file
  3. Go to ChatGPT, Claude, or another AI assistant of your choice
  4. Paste the prompt into the AI chat
  5. Upload utils.py and genai.py files to the AI (if the AI supports file uploads)
  6. The AI will generate code for app.py
  7. Copy the generated code
  8. Create a new file named app.py in your project folder in VS Code
  9. Paste the code into this file and save it

6. Run the Streamlit App

  1. Open a terminal in VS Code:
    • Select Terminal > New Terminal from the menu
  2. In the terminal, run: 
    streamlit run app.py
  3. Alternatively, you can enable automatic reloading when you edit the code (optional): 
    streamlit run app.py --server.runOnSave=true

    With this option, whenever you make changes to your code and save, the app will automatically reload with your changes.

  4. Your app should automatically open in your default web browser
  5. If it doesn't open automatically, you can access it at http://localhost:8501

7. Deploy Online (Optional)

  1. Make sure your code is pushed to GitHub: 
    git add .
git commit -m "Initial commit"
git push
  2. Go to Streamlit Cloud
  3. Sign in with your GitHub account
  4. Click "New app"
  5. Select your repository, branch (usually "main"), and specify the path to the app file (app.py)
  6. Click "Deploy"
  7. Once deployed, go to the app settings
  8. Add your OpenAI API key as a secret:
    • Go to "Secrets" in the app settings
    • Add your OpenAI API key in TOML format: 
      OPENAI_API_KEY = "your_openai_api_key_here"
  9. Your app is now available online and can be shared with others!

Troubleshooting

Common Issues:

  • FFmpeg not found error: Make sure FFmpeg is properly installed and added to your PATH.
  • API key errors: Check that your .env file contains the correct OpenAI API key.
  • Memory errors during video processing: Try with smaller video files if you encounter memory issues.
  • Module not found errors: Ensure all dependencies are installed via pip install -r requirements.txt.

Using the App

Once your app is running, you can:

  1. Upload a video file
  2. Enter instructions for the voiceover style/content
  3. Generate and edit the voiceover text
  4. Select a voice and generate the audio
  5. Adjust the volume levels and merge with your video
  6. View and download the final video with voiceover
